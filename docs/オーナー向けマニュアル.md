# LINE bot + AI自動応答ツール オーナー様向けマニュアル

## 1. はじめに

このツールは、LINEで届くお問い合わせのうち、よくある質問（営業時間・料金・アクセスなど）にAIが自動で回答するものです。

**できること**
- 登録したFAQに基づいて、AIが自動でお客様に返信します
- AIが回答できない質問は、オーナー様のLINEに通知されます（返信はオーナー様ご自身のLINE公式アカウントアプリから行います）
- スマホの管理画面から、FAQの追加・編集や、友だち全員へのお知らせ配信ができます

**できないこと（対象外）**
- 予約受付
- 顧客情報の管理（顧客カルテ）
- 決済

## 2. LINE Developersコンソールの設定手順

このツールを動かすには、LINE公式アカウントの「チャネルシークレット」と「チャネルアクセストークン」が必要です。以下の手順で発行し、管理画面に登録してください。

1. [LINE Developers](https://developers.line.biz/console/) にログインし、対象のチャネル（LINE公式アカウント）を選択する
2. 「Messaging API設定」タブを開く
3. **チャネルシークレット**: 「チャネル基本設定」タブに表示されている値をコピーする
4. **チャネルアクセストークン**: 「Messaging API設定」タブの「チャネルアクセストークン（長期）」の「発行」ボタンを押し、表示された値をコピーする（この値は一度しか表示されないので、必ずメモしておく）
5. 同じ画面の「Webhook設定」で、Webhook URLに次のURLを入力する
   ```
   https://（納品時に案内するドメイン）/api/line/webhook
   ```
6. 「Webhookの利用」を **オン** にする
7. 「検証」ボタンを押し、成功することを確認する

### 重要: デフォルト応答をオフにする

LINE公式アカウントマネージャー（https://manager.line.biz/）にログインし、「応答設定」から以下を **オフ** にしてください。オンのままだと、LINE標準のあいさつメッセージやデフォルト応答がbotの返信と競合し、お客様に二重のメッセージが届いてしまいます。

- あいさつメッセージ
- 応答メッセージ（デフォルト応答）

「Webhook」だけを **オン** の状態にしてください。

### 管理画面への貼り付け

1. 管理画面 (`/admin`) にログインする
2. 「設定」タブを開く
3. 「チャネルシークレット」「チャネルアクセストークン」を貼り付けて保存する

## 3. Claude APIキーの取得と設定

1. [Anthropic Console](https://console.anthropic.com/) にアカウント登録する
2. 「API Keys」からAPIキーを新規発行する
3. 管理画面の「設定」タブにある「Claude API設定」にキーを貼り付けて保存する

## 4. オーナー様のLINE ID登録

AIが答えられない質問があったとき、オーナー様ご自身のLINEに通知するための設定です。

1. 管理画面の「設定」タブを開く
2. 「オーナー様のLINE ID登録」の「コードを発行する」を押す
3. 表示された6桁のコードを、**オーナー様ご自身のLINEアカウントから、この公式アカウントのトーク画面にそのまま送信する**
4. 「設定済み」と表示されれば完了

以降、AIが答えられない質問があると、このLINEアカウントに通知が届くようになります。

## 5. 管理画面の日常的な使い方（スマホから）

管理画面はスマホのブラウザからそのまま操作できます。ホーム画面に追加しておくと便利です。

### ログイン

納品時にお伝えしたIDとパスワードでログインしてください。

### ダッシュボード

- 本日の問い合わせ数、自動応答率、要対応件数が一目でわかります
- 「要対応の質問」には、AIが答えられなかった質問が一覧表示されます（お客様のLINE名が取得できた場合は、質問の横に表示されます）
- **実際のお客様への返信は、LINE公式アカウントアプリから行ってください。** 管理画面の「対応済みにする」ボタンは、社内での対応記録用のチェックであり、お客様への返信機能ではありません

**返信の探し方**: オーナー様のLINEに届く通知には、お客様のLINE名（表示名）が含まれています。LINE公式アカウントアプリのトーク一覧を開き、その名前のトークを探して直接返信してください。名前が表示されない場合（お客様がLINE側で表示名を公開していない等）は、通知が届いた時刻に最も近い新着メッセージを開いてください。

### FAQの追加・編集・削除

- 「FAQ」タブから、よくある質問と回答を登録できます
- 「有効」にしたFAQのみがAIの回答対象になります。一時的に使わないFAQは「無効」にできます
- 「上へ」「下へ」で表示順を調整できます

### お知らせ配信

- 「配信」タブから、友だち全員にメッセージを一斉送信できます
- **送信すると取り消せません。** 内容をよく確認してから送信してください

## 6. トラブルシューティング

### botが反応しない

以下の順に確認してください。

1. LINE公式アカウントマネージャーの「応答設定」で、Webhookがオンになっているか
2. LINE Developersコンソールの「Webhook設定」で、URLが正しく設定され「検証」が成功するか
3. 管理画面の「設定」でチャネルアクセストークンが有効期限切れになっていないか（トークンは再発行して貼り直せます）
4. Vercelのステータス（サービス障害が起きていないか）

### 通知（プッシュ）が来ない

- 管理画面の「設定」→「オーナー様のLINE ID登録」が「設定済み」になっているか確認する
- スマホの機種変更などで届かなくなった場合は、「登録をリセットする」→再度コードを発行し、新しい端末から送信し直す

### 連絡先

不明な点があれば、納品時にお伝えした連絡先までご連絡ください。

## 7. 料金について

- Claude APIは従量課金です（使った分だけ発生します）
- 1日10〜15件程度の問い合わせであれば、月額の目安は概算 2,000〜3,000円程度です
- 使用量はAnthropic Console（https://console.anthropic.com/）の「Usage」ページで確認できます
- 想定より問い合わせ件数が大きく増えた場合は、費用が変動する可能性があります

## 8. 付録

### ログイン情報の保管

- 管理画面のID・パスワード、LINEチャネルシークレット・アクセストークン、Claude APIキーは、いずれも第三者に知られると悪用される可能性がある重要な情報です。メモを残す場合は、パスワード管理アプリなど安全な場所に保管してください

### 用語集

| 用語 | 説明 |
| --- | --- |
| Webhook | LINEに届いたメッセージを、このツールのサーバーに転送するための仕組み |
| チャネルシークレット | LINEからの通知が本物かどうかを確認するための鍵 |
| チャネルアクセストークン | このツールがLINEにメッセージを送るための鍵 |
| FAQ | よくある質問と回答のセット。AIはこの内容だけを根拠に回答する |
| エスカレーション | AIが答えられず、オーナー様に対応を引き継ぐこと |
